## Le Stratège, le Hacker et le Guerrier

Il y a 3 archétypes dans Tennis Tactique.

### Le Stratège

**Gagner sans combattre**

- Analyse avant d'agir
- Trouve les faiblesses
- Utilise la géométrie
- Ne force jamais

**Caractéristiques :**

- Calme
- Patient
- Observateur
- Tactique

### Le Hacker

**Respecter la règle du 80/20**

- Travaille smart, pas hard
- Trouve le hack
- Optimize l'effort
- Efficace

**Caractéristiques :**

- Créatif
- Ingénieux
- Innovant
- Malin

### Le Guerrier

**Se découvrir au combat**

- Donne tout sur le terrain
- Ne lâche rien
- Fight until the end
- Passion

**Caractéristiques :**

- Combatif
- Persévérant
- Brave
- Passionné

### Le Guerrier Pacifique

Le 4e archétype : Combine les 3 :

- Stratège dans la préparation
- Hacker dans l'exécution
- Guerrier dans le combat

### Comment Identifier Ton Archétype

1. **Comment joues-tu ?**
   - Calculé → Stratège
   - Intuitif → Hacker
   - Combatif → Guerrier

2. **Comment perds-tu ?**
   - Calmement → Stratège
   - Avec surprise → Hacker
   - Avec frustration → Guerrier

3. **Comment progresses-tu ?**
   - En analysant → Stratège
   - En testant → Hacker
   - En répétant → Guerrier

### Comment Utiliser Cette Connaissance

- Connais ton archétype
- Travaille tes faiblesses
- Capitalise sur tes forces
- Deviens un Guerrier Pacifique
