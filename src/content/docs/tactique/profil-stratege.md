## Le Profil : Le Stratège

### Caractéristiques

- Calme
- Patient
- Observateur
- Tactique
- Analytique

### Comment le reconnaître

- Il analyse avant d'agir
- Il prend son temps
- Il cherche vos faiblesses
- Il joue le pourcentage
- Il ne force jamais

### Ses Points Forts

- Lecture du jeu
- Adaptation tactique
- Gestion du risque
- Regularité
- Persévérance

### Ses Points Faibles

- Manque parfois d'agressivité
- Peut être prévisible
- Difficulté sous pression

### Comment le Battre

- Acceleration le rythme
- Varier le jeu
- Le bouger dans tous les sens
- Jouer sur ses nerfs
- Sortir de sa zone de confort

### Comment l'Entraîner

- Travail sur l'agressivité
- Exercises de variation
- Scénarios sous pression
- Développement de l'instinct
